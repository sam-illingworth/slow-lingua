# Task-Based Curriculum

Progression is task-based, not grammar-based. Each task teaches language through a real situation.

Tasks unlock adaptively - when you've mastered prerequisites, next tasks become available.

---

## Foundation Tasks (A1)

### Task 1: Talking About Me
**Goal:** Introduce yourself
**Phrases:**
- My name is...
- I live in...
- I am (profession)
- I like...

**Unlocks:** Immediately available

---

### Task 2: Asking About You
**Goal:** Have a basic exchange
**Phrases:**
- What's your name?
- Where do you live?
- What do you do?
- Nice to meet you

**Unlocks:** After Task 1

---

### Task 3: Communication Problems
**Goal:** Handle not understanding
**Phrases:**
- I don't understand
- Can you repeat that?
- More slowly, please
- How do you say...?

**Unlocks:** After Task 1 (teach early!)

---

### Task 4: Numbers and Basics
**Goal:** Count, tell time, basic quantities
**Phrases:**

- What time is it?
- How much does it cost?

**Unlocks:** After Task 2

---

### Task 5: Talking About Family and Friends
**Goal:** Describe people you know
**Phrases:**
- This is my...
- He/she is...
- I have... brothers/sisters
- My friend likes...

**Unlocks:** After Task 2

---

### Task 6: My Routine
**Goal:** Describe daily life
**Phrases:**

- In the afternoon, I...

**Unlocks:** After Task 2

## How Tasks Work

Each task includes:
1. **Phrases** - Core language to learn
2. **Vocabulary** - Words needed for this task
3. **Grammar** - Rules that emerge naturally (taught when encountered)
4. **Exercises** - Speaking, listening, reading, writing activities
5. **Conversation** - Practice using the task in dialogue
6. **Check** - Can you do this task in real life?

Tasks are not "completed" linearly - you return to them for review and deepening.

---

## Skill Weighting Within Tasks

Each task should cover all four skills:
- Speaking: 30%
- Listening: 25%
- Reading: 25%
- Writing: 20%

Adjust based on user's skill balance (more practice on weaker skills).
